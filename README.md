# Atividade Prática - Funções e Manipulação do DOM

## Nome
Maria Luiza Aparecida Trindade de Meneses

## Descrição
Projeto desenvolvido para praticar:
- Manipulação do DOM
- Eventos
- Renderização dinâmica
- Uso de JSON
- Criação de cards
- Filtros e detalhes de produtos

## Funcionalidades
- Renderização dinâmica de produtos
- Busca por texto
- Filtro por categoria
- Destaque visual nos cards
- Área de detalhes do produto

## Métodos utilizados
- getElementById
- querySelector
- querySelectorAll
- createElement
- appendChild
- classList.add
- addEventListener
- innerHTML
